# Learning Path Recommender

A simple system that recommends a personalized learning path based on user skills, level, and career goals.

## Features
- Career-based learning paths
- Skill gap detection
- Interactive web interface using Streamlit

## Tech Stack
- Python
- Pandas
- Streamlit

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py
